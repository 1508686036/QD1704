联想商城

完成的：
1、有首页，详情页，登录，注册，购物车，二级页面

   （1）首页(index.html)：主页中包括商品、二级列表，banner图等

   （2）详情页：有两个，是主页里面 “明星单品”里的前两个，点击即进入详情页（goods1.html  、goods2.html）；
功能：在详情页里面可以加减商品、在商品中有放大镜功能，商品列里可以点击查看该商品的各个图片，加入购物车功能，加入购物车时创建cookie，加入购物车时，详情页上方显示加入商品的数量;

   （3）登录、注册：单例模式，在其他页面中也可以实现；
功能：注册和登录都有验证，注册时创建cookie,登录时从创建的cookie里取数据

   （4）购物车(shopCart.html)：在详情页把商品加入购物车，会跳转到购物车页面；
功能：购物车页面会加载加入的商品的信息，在购物车页面可以对商品的数量、钱数进行加减，点击删除时，可以删除对应的商品，点击选中按钮可以在结算的地方，显示应付商品的钱数；在该页面也可以点击选购商品进入主页面
 
    （5）二级页面(demo.html)：是主页中导航里的“急速达“，点击进入该页面，该页面中主要有轮播图

2、用模块化写的js，数据动态加载，用ajax json引入的数据; 轮播图、首部导航等的动画已加入；
